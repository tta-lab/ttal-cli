## Phase 3 — Mid-scene audit (run every 5 turns)

Quick scan:
- [ ] Object positions consistent across turns?
- [ ] Time of day arc consistent?
- [ ] No voice drift on any character?
- [ ] No mechanism rule contradictions?
- [ ] Any drift away from Resolution paragraph?

If any "no" → out-of-frame correct in next turn (do not silent retcon).

---

## Phase 4 — Closing audit checklist

Before writing the final turn:

- [ ] Resolution check: each character lands at the state described in Resolution paragraph?
- [ ] Character integrity: each character speaking in their established register? (no soft-warm-ifying a hard character for sentiment)
- [ ] Plot stance committed: if scene had an open question, one stance committed (no both-sides-implied wishful ambiguity)?
- [ ] No new actors introduced in close?
- [ ] No new mechanisms introduced in close?
- [ ] Internal continuity audit: re-read all turn commits, no contradictions found?
- [ ] No retcons applied during scene (all corrections were out-of-frame)?
